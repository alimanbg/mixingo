# Mixingo

Language learning that starts from who you are. Mixingo maps your existing languages (English, Cantonese, Mandarin, etc.) to your target language (French) and accelerates learning by up to 32%.

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Local development

```sh
# Install dependencies
npm install

# Start dev server
npm run dev
```

The app runs at `http://localhost:8080` by default.

## Build

```sh
npm run build
npm run preview  # Preview production build
```
